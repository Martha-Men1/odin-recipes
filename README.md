# Odin Recipes 🍽️

This project is a simple multi-page recipe website built as part of *The Odin Project* curriculum. It demonstrates foundational HTML skills by creating a structured website with multiple pages and basic navigation.

## Overview

The website includes a homepage and several recipe pages. Each recipe is organized with clear sections such as description, ingredients, and preparation steps. Users can easily navigate between pages using relative links.

## Features

* **Homepage (index.html)**
  Displays the main title *"Odin Recipes"* and a list of links to individual recipe pages.

* **Recipe Pages**
  Each recipe page contains:

  * Recipe title
  * Image of the finished dish
  * Description of the recipe
  * Ingredients list (unordered list)
  * Step-by-step instructions (ordered list)
  * A link back to the homepage

* **Images Directory**
  Stores all images used across recipe pages for better organization and cleaner file structure.

* **Consistent Layout**
  All recipe pages follow the same structure for readability and maintainability.

## Project Structure

```
odin-recipes/
│── index.html
│── images/
│   ├── lasagna.jpg
│   ├── potato.jpg
│   └── pumpkin.jpg
└── recipes/
    ├── lasagna.html
    ├── potato_salad.html
    └── pumpkin_spice_latte.html
```

## Learning Objectives

This project reinforces:

* Writing standard HTML boilerplate
* Structuring a multi-page website
* Using relative paths for links and images
* Organizing files into directories
* Using lists, headings, images, and links effectively

---

This project serves as a foundational step in learning web development and building more complex, styled websites in the future.
